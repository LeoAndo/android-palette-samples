# Overview
imageViewのBitmapから色を抽出してくれるライブラリ(palette)を使ったサンプルアプリです。<br>
paletteライブラリは1.0リリース後にUpdateがなく、ライブラリ内部でAndroid 11で非推奨APIになった、`AsyncTask`を使っていたりしているので、利用しない方が良いかも<br>
